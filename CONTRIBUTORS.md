## How to Get Listed

### Get Added to This File

1. **Make a meaningful contribution**
   - Code contribution
   - Documentation
   - Bug report/fix
   - Feature request that gets implemented

2. **Open a PR adding yourself**
   - Link to your GitHub profile
   - Brief description of contribution
   - Category (Code, Docs, Testing, etc.)

3. **Get merged and celebrate!** 🎉

### Attribution in Commits

We use [All Contributors Bot](https://allcontributors.org/):

```bash
@allcontributors add @username for code,test,doc
```

This auto-generates contributor recognition.
